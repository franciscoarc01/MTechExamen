# MTechExamen

Es la solución del examen de la empresa MTech para integración con ellos, de lo cual la solución es realizada en .NET con C# como lenguaje de programación.

## El inicio

Para poder iniciar el proyecto es necesario tener Visual Studio para compilar el proyecto

### Prerequisitos

Tener Visual Studio

### Solución

La solución fue con web API de .NET para resolver el examen.

En el mismo viene las pruebas unitarias de la solución.
